VSLogcat
========

simple logcat for visual studio

Install
======
1,build this project ,install "VSLogcat.vsix" ,then restart your visual studio;

2,add file "Log.cs" to your project;

Example
======
Log.I("info-------------------->info");

Log.D("debug-------------------->debug");

Log.W("warn-------------------->warn");

Log.E("error-------------------->error");

Log.V("verbose-------------------->verbose");
